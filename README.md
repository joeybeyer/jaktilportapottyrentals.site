## Project Overview

Jaktil Porta Potty Rentals is a hyper-optimized local lead generation property designed to dominate AI Overviews, Google Maps, and organic search for portable toilet rental queries. The site leverages cutting-edge SEO strategies including entity stacking, BERT optimization, and the Research Company method to achieve rapid ranking and sustained visibility.

### Core Philosophy

```
Performance + Structure + Signal Clarity = AI Era Ranking
```

**Key Insight:** Physical distance is only 3% of the proximity equation. Relevance score overrides distance decay.

### Critical Rule

**NEVER create City + Service combo pages:**
- ❌ `/milwaukee-porta-potty-rental/`
- ✅ `/milwaukee/` (city hub) + `/porta-potty-rental/` (service page)

## Content Strategy

### BERT Optimization Hierarchy

| Element | Confidence Level | Implementation |
|---------|------------------|----------------|
| Tables | EXTREME | Pricing tables, size comparisons |
| Unordered Lists | Very High | Quick summary bullets at TOP |
| H2/H3 Headings | High | Service + Geo within 2 tokens |
| Definition Lists | High | FAQ schema opportunities |

### Information Gain (80% Unique Rule)

Each neighborhood page must include:
- Local event venues and landmarks
- Permit requirements for that area
- Accessibility considerations
- Community-specific FAQs
- Local pricing factors

---

## Internal Linking Formula

```
1 UP + 2-3 ACROSS = Money
```

- **1 UP:** Link to City Hub (first paragraph, above fold)
- **2-3 ACROSS:** Link to Non-Geo Service Pages
- **2-3 ACROSS:** Link to NEIGHBORING neighborhoods (not all)

### Anchor Text Distribution

| Type | Percentage | Example |
|------|------------|---------|
| Partial Match | 40-50% | "our portable toilet services" |
| Branded | 20-30% | "Jaktil team" |
| Natural Phrase | 15-20% | "learn how we serve events" |
| Exact Match | 5-10% | "porta potty rental" (sparingly) |
| Generic | 0% | Never use "click here" |
